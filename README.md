# Game Scout API

API de recomendacion de videojuegos orientada a integraciones de terceros.

## Overview

Game Scout API expone endpoints para descubrir videojuegos, gestionar perfiles/preferencias de usuario y obtener recomendaciones personalizadas con explicaciones.

## Features (MVP)

- Catalogo de juegos con filtros y detalle por juego.
- Perfil de usuario con preferencias e interacciones.
- Recomendaciones personalizadas con explicabilidad.
- Autenticacion por API key para clientes.
- Control de uso por rate limit y cuotas.

## Tech Stack (Baseline)

- Python 3.12
- FastAPI
- PostgreSQL + pgvector
- Redis
- Docker / Docker Compose
