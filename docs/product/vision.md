# Product Vision

## Problem

Teams and developers need an API-first way to get personalized game recommendations with explainable ranking.

## Objective

Build a public API product that:

- Recommends games with a hybrid engine (content + semantic + rules)
- Exposes recommendations through API keys
- Supports usage plans with rate limits and quotas
- Keeps code open while monetizing operation, reliability, and support

## MVP Scope

- Catalog ingestion and cleaning
- User preferences and interactions
- Recommendation endpoint with reasons
- API key auth
- Plan limits (rate, daily quota, monthly quota)
- Usage tracking by client and endpoint

## Non-MVP

- Full billing automation
- Advanced overage logic
- Full self-service customer portal
